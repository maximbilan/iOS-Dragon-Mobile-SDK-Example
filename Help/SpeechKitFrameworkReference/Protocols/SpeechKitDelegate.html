SpeechKitDelegate protocol reference

SpeechKit.h

Overview

The SpeechKitDelegate protocol defines the messages sent to a delegate object registered as part of the call to setupWithID.

Tasks

Observing state

  • - destroyed Sent when the destruction process is complete.

Instance methods

destroyed

Sent when the destruction process is complete.

- (void) destroyed
Discussion

This allows the delegate to monitor the destruction process. Note that subsequent calls to destroy and setupWithID will be ignored until this delegate method is called, so if you need to call setupWithID again to connect to a different server, you must wait for this.
