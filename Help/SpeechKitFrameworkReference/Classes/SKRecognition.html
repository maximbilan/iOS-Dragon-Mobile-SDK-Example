Overview

The SKRecognition object contains recognition results returned from SKRecognizer.

Tasks

Properties

  • results property An NSArray containing NSStrings corresponding to what may have been said, in order of likelihood.
  • scores property An NSArray of NSNumbers containing the confidence scores corresponding to each recognition result in the results array.
  • suggestion property A server-generated suggestion suitable for presentation to the user.

Accessors

  • - firstResult Returns the first NSString result in the results array.

Instance methods

firstResult

Returns the first NSString result in the results array.

- (NSString *)firstResult
Return value

The first NSString result in the results array or nil.

Discussion

This is a convenience method to simply return the best result in the list of results returned from the recognition server. It returns nil if the list of results is empty.

Properties

results

An NSArray containing NSStrings corresponding to what may have been said, in order of likelihood.

@property NSArray *results
Discussion

The speech recognition server returns an array of recognition results by order of confidence. The first result in the array is what was most likely said. The remaining list of alternatives is ranked from most likely to least likely.

suggestion

A server-generated suggestion suitable for presentation to the user.

@property NSString *suggestion
Discussion

This is a suggestion to the user about how he or she can improve recognition performance and is based on the audio received. Examples include moving to a less noisy location if the environment is extremely noisy, or waiting a bit longer to start speaking if the beeginning of the recording seems truncated. Results are often still present and may still be of useful quality.
