Dragon Mobile SDK¶

The Dragon Mobile SDK provides speech services to developers who want to enhance their applications with speech recognition and text-to-speech functionality. The Dragon Mobile SDK consists of a variety of code examples and sample projects, documentation, and a simple framework to provide an easy path to integrating speech services into any application.

Organization of This Document¶

This document contains the following articles:

  • “Speech Kit Guide” provides details of the Speech Kit framework.
  • “Sample Speech Recognition App” describes the speech recognition sample.
  • “Sample Text-To-Speech App” describes the text-to-speech sample.
  • “Solutions to Common Problems” provides solutions to common problems.
